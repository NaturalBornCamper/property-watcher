# Rental Listings

Accepted rental listings that match the current filters.

***LAST UPDATED:*** **2026-07-01**

## Current filters

- Postal code prefix starts with `H4H` or `H8P`.
- At least 2 rooms or bedrooms.
- At least 900 square feet when a real size is available.
- Missing size, or obvious platform defaults such as `1 sqft`, are treated as `Unknown` rather than rejection reasons.
